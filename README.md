# Curve Foundry

Curve Foundry is a static browser prototype for turning a drawn curve into editable habitat descriptors and designing a related colony species.

## What it does

- `index.html` is the habitat page for coral nests / mothership structures driven by a curve
- `creatures.html` is the creature page for designing the colony species and its swarm behaviour
- Both pages are static, local-first browser prototypes
- Both pages support copying a shareable URL with the current settings embedded in it

## Habitat descriptor set

- `generatorFamily`
- `averageSpeed`
- `peakSpeed`
- `smoothness`
- `jaggedness`
- `directness`
- `turningIntensity`
- `selfCrossings`
- `endEdge`

## Habitat generator families

- `fern-like`
- `conch-like`
- `dodecahedral`
- `fluffy-cloud`
- `cuboid`

## Good test prompts

- Draw a smooth loop and end near the top edge
- Draw a jagged zig-zag and end on the right edge
- Draw a self-crossing scribble with several loops
- Keep the same numbers but swap generator family modes
- Open `creatures.html` and compare a solitary rigid species against a highly social squiggly one
- Use the copy button and compare settings with a friend

## Local use

Open `index.html` or `creatures.html` in a browser.

You can also open either page with a `?state=...` query parameter generated by the copy button to restore a shared setup directly.

## Publishing

This repo is static, so it works well with GitHub Pages from the repository root or `/docs`.
